# Medical AI Evaluation Portfolio

## About This Project

This repository documents my journey in becoming a Medical AI Evaluator.

The project focuses on evaluating AI-generated clinical responses using evidence-based medicine and international clinical guidelines.

## Objectives

- Evaluate medical AI responses.
- Detect hallucinations and factual errors.
- Assess patient safety.
- Compare responses with international guidelines.
- Build a professional portfolio for AI healthcare positions.
